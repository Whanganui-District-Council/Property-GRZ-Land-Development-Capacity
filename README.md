# Property-GRZ-Land-Development-Capacity

This repository contains files used to quantify land development capacity within the General Residential Zone (GRZ). 

The model integrates spatial, planning, infrastructural, and environmental constraints to identify:
*	Vacant development opportunities
*	Infill development potential
*	Fully developed properties

The framework leverages PostGIS spatial analytics, materialized data pipelines, and a rule-based exclusion system to produce consistent, scalable, and policy-aligned outputs.
